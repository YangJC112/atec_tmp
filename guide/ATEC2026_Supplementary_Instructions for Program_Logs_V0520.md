**ATEC2026 Online Competition: Supplementary Instructions for Program Logs**

**1. Log File Path:**
Program logs must be output to the `${LOG_DIR}/user.log` file.

**2. Viewing Method:**
Contestants can view the logs by clicking the **"Log"** button on the **"Submission Records"** page.

**3. Implementation Reference:**
For the specific implementation method, please refer to the `setup_logging()` function in the `server.py` file.

**4. Display Limit:**
The log window will display a maximum of the **most recent 500 lines** of log content.

**Note:**
The `LOG_DIR` environment variable is automatically injected by the competition system; no manual configuration by contestants is required.
